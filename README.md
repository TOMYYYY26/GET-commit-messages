# GET-commit-messages
# 🛠️ GitLab & Gerrit 提交信息提取工具

## 📦 功能概述

| 功能模块       | 描述                                                                 | 输出位置           |
|----------------|----------------------------------------------------------------------|--------------------|
| `gitlab.py`    | 获取 GitLab 仓库完整提交记录（含代码差异）                          | `./diff_report/`   |
| `get.py`       | 获取 Gerrit 仓库基础提交信息（changeID 等，代码内容待实现）         | 控制台输出         |
